# PDF-Tool

The following functions can already be used:

  - Merge 2 or more PDFs into one PDF
  - Delete individual pages from a PDF
  - Protect PDFs with passwords
  - Conversions:
    - .doc, .docx to PDF
    - .xls, .xlsx to PDF
    - .ppt, .pptx to PDF
    - .png, .jpeg to PDF

- editing PDFs
- Rotate PDF pages
